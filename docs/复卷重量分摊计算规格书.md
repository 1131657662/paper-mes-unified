# 复卷重量分摊计算规格书（P0-0）

> 对应《卷筒纸加工管理系统 开发文档 V4.1》§2.6.1。
> 本规格书是全系统计算精度的唯一基准；所有重量分摊、修边扣减、尾差消除以此为准。
> 状态：Phase 1 前置交付物。**复卷相关计费/回录编码（Phase 2 P1-5）开工前必须以本文档为准实现并通过下文用例单测。**

---

## A. 基础换算与符号约定

| 符号 | 含义 | 来源 |
|------|------|------|
| `W_actual` | 原纸实际重量 (kg) | `biz_original_roll.actual_weight` |
| `D_out` | 成品外径 (mm) | `finish_diameter` 英寸换算 |
| `D_core` | 成品纸芯直径 (mm) | `finish_core_diameter` 英寸换算 |
| `width` | 门幅 (mm) | `finish_width` |
| `N` | 成品件数 | — |
| `S_i` | 第 i 件横截面积 (mm²) | 计算值 |
| `trim_weight_share_i` | 第 i 件分摊修边重量 (kg) | 计算值 |

**英寸→毫米换算：`1 英寸 = 25.4 mm`。** 直径、纸芯直径录入为英寸，内部统一换算为 mm 后参与面积计算。

实现约定：
- 换算 `mm = inch × 25.4`，用 `BigDecimal`，不用 double 累计。
- 圆周率统一 `Math.PI`；面积中间值保留至 mm²，最终重量四舍五入到 **3 位小数**（kg，对齐 `decimal(10,3)`）。
- 因 π 精度，与本文档手算期望值允许 **±0.5 kg** 容差（单测断言用容差比较，不做精确相等）。

---

## B. 纸卷横截面积公式

```
S = π × ( (D_out/2)² − (D_core/2)² )      单位 mm²
```

建议方法：`BigDecimal crossSectionArea(BigDecimal dOutMm, BigDecimal dCoreMm)`

当模式只改门幅、外径与纸芯一致时各件 S 相等，面积占比退化为门幅占比（即模式1）。

---

## C. 单件重量分摊公式（通用式，模式 1~4）

```
S_total = Σ S_i
weight_i = W_actual × ( S_i / S_total ) − trim_weight_share_i
```

各模式 `S_i` 取法：
- **模式1 改门幅不变直径**：`S_i ∝ width_i`，面积占比即门幅占比（S_total 直接用门幅之和）。
- **模式2 改直径不变门幅**：`S_i` 按 B 式横截面积。
- **模式3 改直径+改门幅**：`S_i = 横截面积_i × (width_i / 原纸门幅)`。
- **模式4 内外层分层**：每层独立按 B 式算面积并入 S_total，分层参数存 `biz_process_param`。
- **模式5 多母卷合并复卷**：不按面积，按自定义比例 `ratio_i`（Σratio_i=1）：
  `weight_i = (Σ各母卷 W_actual) × ratio_i − trim_weight_share_i`。

---

## D. 修边分摊公式

```
trim_weight_total = ( 总修边宽度 / 原纸门幅 ) × W_actual
trim_weight_share = trim_weight_total / N
```

总修边宽度 ≤ 1mm 时执行 §2.8 修边豁免，`trim_weight_total` 一键清零。

---

## E. 尾差倒挤规则（消除小数累计误差）

```
weight_N = W_actual − Σ(前 N−1 件 weight_i) − 总损耗重量 total_loss_weight
```

最后一件不走 C 式分摊，由整卷余量倒挤，保证整卷重量严格闭合（对接 §5.7 三级校验）。

---

## F. 完整用例（单元测试基准）

### 用例1 — 模式2 改直径不变门幅（无修边、无损耗）

**输入**
- `W_actual = 1000.000 kg`，门幅 1200mm
- 原纸外径 40 英寸 = 1016mm，纸芯 3 英寸 = 76.2mm
- 拆 2 件，门幅不变：A 外径 30 英寸=762mm；B 外径 28.28 英寸≈718.3mm；纸芯均 76.2mm

**中间值**
- `S_A = π×((762/2)²−(76.2/2)²) = π×(145161−1451.61) ≈ 451471 mm²`
- `S_B = π×((718.3/2)²−(38.1)²) = π×(128988.6−1451.61) ≈ 400666 mm²`
- `S_total ≈ 852137 mm²`

**期望输出**
- `weight_A = 1000 × (451471/852137) ≈ 529.81 kg`
- `weight_B`（最后一件倒挤）`= 1000 − 529.81 − 0 = 470.19 kg`
- **闭合校验：529.81 + 470.19 = 1000.00 ✔**

### 用例2 — 模式1 改门幅不变直径 + 修边

**输入**
- `W_actual = 800.000 kg`，门幅 1500mm
- 拆 3 件，门幅 500 / 500 / 480mm，总修边宽度 20mm

**中间值**
- `trim_weight_total = (20/1500)×800 = 10.667 kg`；单件修边 `= 10.667/3 ≈ 3.556 kg`
- 门幅占比基准 `S_total = 500+500+480 = 1480`（修边 20mm 不计入成品门幅）

**期望输出**
- `weight_1 = 800×(500/1480) − 3.556 = 270.270 − 3.556 = 266.714 kg`
- `weight_2 = 800×(500/1480) − 3.556 = 266.714 kg`
- `weight_3`（最后一件倒挤，总损耗=修边总重）`= 800 − 266.714 − 266.714 − 10.667 = 255.905 kg`
- **闭合校验：266.714 + 266.714 + 255.905 + 10.667 = 800.000 ✔**

---

## G. 重量闭合优先原则（实称值优先于公式）

当某件成品已有实称重量（`biz_finish_roll.actual_weight` 非空）时：
1. 该件**不走 C 式面积分摊**，直接取实称值参与整卷重量。
2. 面积分摊仅作为**剩余未实称件**的权重：`S_total` 仅累计未实称件面积，`W_actual` 先扣除已实称件重量后再分摊。
3. 最后一件仍执行 E 式尾差倒挤，**以原纸实际总重为唯一闭合基准**，吸收全部尾差。

人工实测数据永远优先于系统理论估算；系统估算只填补尚未实测的空缺。

## 实现与测试备注（供 Phase 2 P1-5）

- 计算入口建议独立为 `RewindWeightCalculator`（无状态、纯函数），便于单测。
- 用例1/用例2 直接转 JUnit：给定输入 → 断言每件 weight（±0.5kg 容差）→ 断言整卷闭合（Σweight + Σ损耗 == W_actual，精确相等）。
- 闭合校验是硬约束：尾差倒挤后整卷必须严格等于 `W_actual`，对接 §5.7 三级重量校验。
